# EduSpace - RIT Capstone Project

Frontend portion of the Eduspace RIT Capstone Project

## Frontend Tools
- [React](https://react.dev/): JS Library for Web Apps and User Interfaces
- [Vite](https://vite.dev/): Build tool for React Projects
- [Docker](https://www.docker.com/): Container system to run our frontend environment
- [ThreeJS](https://threejs.org/): 3D Animation Library for JS
- [React Three Fiber](https://r3f.docs.pmnd.rs/getting-started/introduction): React Renderer for Three.js

## Prerequisites
1. Install Docker Desktop: https://docs.docker.com/desktop/
2. Install Node v20.20.2 and above: https://nodejs.org/en/download

## Setup InstructionsS
1. cd into the `eduspace-simulation` directory. Install the necessary packages using `npm install`

2. To start a local instance of the simulation, type `npm dev`. Then, go to the link provided by vite. Since vite supports hot reloading, any update saved in the codebase will be compiled and deployed in the aformentioned link.
